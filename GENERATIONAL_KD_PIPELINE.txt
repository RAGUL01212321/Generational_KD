================================================================================
GENERATIONAL KNOWLEDGE DISTILLATION PIPELINE - PSEUDO CODE
================================================================================

Extended Apollo KD pipeline for multi-generation student distillation
- Generation 0: Qwen1.5-1.8B (Teacher, frozen)
- Generation 1: Qwen1.5-0.5B (Student 1)
- Generation 2: Smaller model (Student 2, learns from Gen 0 + Gen 1)
- Generation N: ... and so on

Each successive generation learns from ALL predecessors in shared projection space,
accumulating knowledge through the lineage.

================================================================================
2. MULTI-GENERATION MODEL INITIALIZATION
================================================================================

InitializeGenerationalModels(generational_params):
    
    models = {}          // M[0], M[1], M[2], ...
    projections = {}     // P[0], P[1], P[2], ...
    optimizers = {}      // One optimizer per student
    
    FOR each generation_config IN generational_params.generation_models:
        gen_id = generation_config.id
        model_id = generation_config.model
        is_frozen = generation_config.frozen
        
        IF gen_id == 0:
            // Load teacher model (Generation 0)
            models[0] = LOAD_PRETRAINED(model_id, dtype=float32)
            models[0].eval()
            FREEZE all models[0].parameters()
            
            // Create teacher projection layer
            projections[0] = ProjectionLayer(
                input_dim=2048,          // Qwen1.8B hidden dim
                output_dim=768           // Common projection dim
            )
            projections[0].eval()
            FREEZE projections[0].parameters()
        
        ELSE:
            // Load student models (Generation 1+)
            models[gen_id] = LOAD_PRETRAINED(model_id, dtype=float32)
            IF is_frozen:
                models[gen_id].eval()
                FREEZE models[gen_id].parameters()
            ELSE:
                models[gen_id].train()
            
            // Create student projection layer
            IF model_id == "Qwen/Qwen1.5-0.5B":
                input_dim = 1024
            ELSE IF model_id == "Qwen/Qwen1.5-0.4B":
                input_dim = 896
            // ... etc
            
            projections[gen_id] = ProjectionLayer(
                input_dim=input_dim,
                output_dim=768               // Same common dim
            )
            projections[gen_id].train()
            
            // Create optimizer for this generation
            optimizers[gen_id] = Adafactor(
                params=list(models[gen_id].parameters()) + 
                        list(projections[gen_id].parameters()),
                lr=learning_rate,
                scale_parameter=True,
                relative_step=True
            )
    
    RETURN models, projections, optimizers


================================================================================
3. PROJECTION MECHANISM (Shared Across Generations)
================================================================================

ProjectionLayer(hidden_states, input_dim, output_dim):
    // All generations project to common_dim = 768
    // Allows comparison between vastly different hidden dimensions
    // Example: 2048 (teacher) → 1024 (student1) → 896 (student2) → all 768
    
    projected = LINEAR(hidden_states, input_dim, output_dim)
    RETURN projected


================================================================================
4. GENERATIONAL LOSS COMPUTATION
================================================================================

ComputeGenerationalLoss(
    current_gen_id,
    models,
    projections,
    current_student_hidden,
    current_student_logits,
    all_previous_hidden,
    labels,
    attention_mask,
    loss_weights
):
    
    // ===== KNOWLEDGE DISTILLATION LOSS FROM ALL PREDECESSORS =====
    // Current student learns from ALL previous generations (0...k-1)
    
    kd_loss = 0
    
    FOR pred_gen_id IN range(0, current_gen_id):
        // Get predecessor hidden states (already computed, no_grad)
        pred_hidden = all_previous_hidden[pred_gen_id]
        
        // Project both to common space
        Z_current = projections[current_gen_id](current_student_hidden)
        Z_pred = projections[pred_gen_id](pred_hidden)
        
        // Pool with attention mask
        p_current = MEAN_POOL(Z_current, attention_mask)  // (batch, 768)
        p_pred = MEAN_POOL(Z_pred, attention_mask)        // (batch, 768)
        
        // Weighted MSE loss to each predecessor
        gen_weight = loss_weights.get(pred_gen_id, 1.0 / current_gen_id)
        kd_loss_to_pred = gen_weight * MSE(p_current, p_pred)
        
        kd_loss += kd_loss_to_pred
        
        LOG(f"  Gen{current_gen_id}→Gen{pred_gen_id} KD Loss: {kd_loss_to_pred:.6f}")
    
    END FOR
    
    
    // ===== CROSS-ENTROPY LOSS (language modeling anchor) =====
    shift_logits = current_student_logits[..., :-1, :]
    shift_labels = labels[..., 1:]
    shift_mask = attention_mask[..., 1:]
    
    ce_loss_per_token = CROSS_ENTROPY(shift_logits, shift_labels, reduction=none)
    masked_ce_loss = ce_loss_per_token * shift_mask
    ce_loss = masked_ce_loss.sum() / shift_mask.sum()
    
    
    // ===== COMBINED LOSS =====
    total_loss = (kd_loss_weight * kd_loss) + (ce_loss_weight * ce_loss)
    
    RETURN total_loss, kd_loss, ce_loss


================================================================================
5. GENERATIONAL TRAINING LOOP
================================================================================

TrainGenerationalEpoch(
    current_gen_id,
    models,
    projections,
    optimizers,
    dataloader,
    loss_weights,
    epoch_num
):
    accumulated_loss = 0
    accumulated_kd_loss = 0
    accumulated_ce_loss = 0
    num_steps = 0
    
    FOR EACH batch IN dataloader:
        input_ids = batch["input_ids"]
        attention_mask = batch["attention_mask"]
        
        // ===== FORWARD PASS: ALL PREVIOUS GENERATIONS (NO GRAD) =====
        all_previous_hidden = {}
        
        WITH torch.no_grad():
            FOR pred_gen_id IN range(0, current_gen_id):
                pred_output = models[pred_gen_id](
                    input_ids=input_ids,
                    attention_mask=attention_mask,
                    output_hidden_states=True
                )
                pred_hidden = pred_output.hidden_states[-1]
                all_previous_hidden[pred_gen_id] = pred_hidden
            
            END FOR
        
        
        // ===== FORWARD PASS: CURRENT GENERATION (WITH GRAD) =====
        current_output = models[current_gen_id](
            input_ids=input_ids,
            attention_mask=attention_mask,
            output_hidden_states=True
        )
        current_hidden = current_output.hidden_states[-1]
        current_logits = current_output.logits
        
        
        // ===== LOSS COMPUTATION (from all predecessors) =====
        total_loss, kd_loss, ce_loss = ComputeGenerationalLoss(
            current_gen_id=current_gen_id,
            models=models,
            projections=projections,
            current_student_hidden=current_hidden,
            current_student_logits=current_logits,
            all_previous_hidden=all_previous_hidden,
            labels=input_ids,
            attention_mask=attention_mask,
            loss_weights=loss_weights
        )
        
        // Scale loss by gradient accumulation
        scaled_loss = total_loss / gradient_accumulation_steps
        
        
        // ===== BACKWARD PASS =====
        scaled_loss.backward()
        
        accumulated_loss += total_loss.item()
        accumulated_kd_loss += kd_loss.item()
        accumulated_ce_loss += ce_loss.item()
        num_steps += 1
        
        
        // ===== OPTIMIZER STEP =====
        IF num_steps % gradient_accumulation_steps == 0:
            CLIP_GRAD_NORM(models[current_gen_id].parameters(), max_norm=1.0)
            optimizers[current_gen_id].step()
            optimizers[current_gen_id].zero_grad()
            
            IF num_steps % log_every == 0:
                avg_loss = accumulated_loss / num_steps
                LOG(f"Gen{current_gen_id} Epoch {epoch_num}, Step {num_steps}:")
                LOG(f"  Total Loss: {avg_loss:.4f}")
                LOG(f"  KD Loss: {accumulated_kd_loss/num_steps:.4f}")
                LOG(f"  CE Loss: {accumulated_ce_loss/num_steps:.4f}")
        
        
        // ===== CHECKPOINTING =====
        IF num_steps % save_every == 0:
            SAVE_CHECKPOINT(
                generation=current_gen_id,
                model=models[current_gen_id],
                projections=projections,
                optimizer=optimizers[current_gen_id],
                epoch=epoch_num,
                step=num_steps
            )
    
    END FOR
    
    RETURN epoch_metrics


TrainAllGenerations(models, projections, optimizers, dataloader, num_generations):
    
    FOR gen_id IN range(1, num_generations):
        LOG("="*80)
        LOG(f"TRAINING GENERATION {gen_id}")
        LOG("="*80)
        LOG(f"Learning from {gen_id} predecessor(s):")
        
        FOR pred_id IN range(0, gen_id):
            LOG(f"  - Generation {pred_id}")
        
        FOR epoch IN range(num_epochs):
            LOG(f"\n--- Epoch {epoch}/{num_epochs} ---")
            
            epoch_metrics = TrainGenerationalEpoch(
                current_gen_id=gen_id,
                models=models,
                projections=projections,
                optimizers=optimizers,
                dataloader=dataloader,
                loss_weights=compute_loss_weights(gen_id),
                epoch_num=epoch
            )
            
            LOG(f"Generation {gen_id}, Epoch {epoch}:")
            LOG(f"  Average Loss: {epoch_metrics['avg_loss']:.4f}")
            LOG(f"  Average KD Loss: {epoch_metrics['avg_kd_loss']:.4f}")
            LOG(f"  Average CE Loss: {epoch_metrics['avg_ce_loss']:.4f}")
            LOG(f"  Training Time: {epoch_metrics['time']:.2f} minutes")
        
        SAVE_CHECKPOINT(models[gen_id], f"generation_{gen_id}_final.pt")
    
    END FOR


compute_loss_weights(current_gen_id):
    // Equal weight distribution among all predecessors
    // Could be modified to weight recent predecessors more heavily
    
    IF current_gen_id == 0:
        RETURN {}
    
    weight_per_predecessor = 1.0 / current_gen_id
    weights = {}
    
    FOR pred_id IN range(0, current_gen_id):
        weights[pred_id] = weight_per_predecessor
    
    RETURN weights


================================================================================
6. GENERATIONAL VALIDATION PHASE
================================================================================

ValidateGenerationalStudent(
    gen_id,
    models,
    projections,
    all_models_up_to_gen,
    validation_dataloader
):
    // Evaluate current generation against all predecessors
    
    models[gen_id].eval()
    
    current_metrics = {
        "kd_losses": {},      // KD loss to each predecessor
        "ce_loss": 0,
        "cosine_similarities": {}
    }
    
    total_ce_loss = 0
    num_batches = 0
    
    WITH torch.no_grad():
        FOR EACH batch IN validation_dataloader:
            input_ids = batch["input_ids"]
            attention_mask = batch["attention_mask"]
            
            // Current generation forward
            current_output = models[gen_id](
                input_ids=input_ids,
                attention_mask=attention_mask,
                output_hidden_states=True
            )
            current_hidden = current_output.hidden_states[-1]
            current_logits = current_output.logits
            
            
            // Compare to all predecessors
            FOR pred_id IN range(0, gen_id):
                pred_output = models[pred_id](
                    input_ids=input_ids,
                    attention_mask=attention_mask,
                    output_hidden_states=True
                )
                pred_hidden = pred_output.hidden_states[-1]
                
                // Project
                Z_current = projections[gen_id](current_hidden)
                Z_pred = projections[pred_id](pred_hidden)
                
                // Pool
                p_current = MEAN_POOL(Z_current, attention_mask)
                p_pred = MEAN_POOL(Z_pred, attention_mask)
                
                // Metrics
                kd_loss = MSE(p_current, p_pred)
                cosine_sim = COSINE_SIMILARITY(p_current, p_pred, dim=-1).mean()
                
                IF pred_id NOT IN current_metrics["kd_losses"]:
                    current_metrics["kd_losses"][pred_id] = 0
                    current_metrics["cosine_similarities"][pred_id] = 0
                
                current_metrics["kd_losses"][pred_id] += kd_loss
                current_metrics["cosine_similarities"][pred_id] += cosine_sim
            
            
            // CE loss
            shift_logits = current_logits[..., :-1, :]
            shift_labels = input_ids[..., 1:]
            shift_mask = attention_mask[..., 1:]
            
            ce_loss_per_token = CROSS_ENTROPY(shift_logits, shift_labels, reduction=none)
            masked_ce_loss = ce_loss_per_token * shift_mask
            ce_loss = masked_ce_loss.sum() / shift_mask.sum()
            
            total_ce_loss += ce_loss
            num_batches += 1
    
    END WITH
    
    // Average metrics
    FOR pred_id IN current_metrics["kd_losses"]:
        current_metrics["kd_losses"][pred_id] /= num_batches
        current_metrics["cosine_similarities"][pred_id] /= num_batches
    
    current_metrics["ce_loss"] = total_ce_loss / num_batches
    
    RETURN current_metrics


CompareGenerations(models, projections, validation_dataloader, num_generations):
    LOG("\n" + "="*80)
    LOG("GENERATIONAL VALIDATION RESULTS")
    LOG("="*80 + "\n")
    
    all_gen_metrics = {}
    
    FOR gen_id IN range(0, num_generations):
        IF gen_id == 0:
            LOG(f"Generation {gen_id} (Teacher - Reference)")
            // Teacher metrics computed separately if needed
        ELSE:
            LOG(f"\nGeneration {gen_id}:")
            
            metrics = ValidateGenerationalStudent(
                gen_id=gen_id,
                models=models,
                projections=projections,
                all_models_up_to_gen=models,
                validation_dataloader=validation_dataloader
            )
            
            all_gen_metrics[gen_id] = metrics
            
            LOG(f"  CE Loss: {metrics['ce_loss']:.4f}")
            LOG(f"  Cross-generational KD losses:")
            
            FOR pred_id IN sorted(metrics["kd_losses"].keys()):
                kd_loss = metrics["kd_losses"][pred_id]
                cos_sim = metrics["cosine_similarities"][pred_id]
                LOG(f"    ← Gen{pred_id}: KD Loss={kd_loss:.4f}, Cosine Sim={cos_sim:.4f}")
    
    RETURN all_gen_metrics

MainGenerationalPipeline():

    LOG("="*80)

    LOG("STEP 3: Initializing generational models...")
    models, projections, optimizers = InitializeGenerationalModels(generational_params)
    
    LOG(f"  ✓ Initialized {len(models)} model(s):")

        LOG(f"    Generation {gen_id}: Frozen={IS_FROZEN(model)}")
    
    // STEP 4: Verify Data Integrity
    LOG("STEP 4: Verify    step=num_steps
            )
    ing data separation...")
    data_integrity_pass = CheckDataIntegrity(
        training_params.dataset_file,

    LOG("\n✓ All generations trained and validated successfully!")

SaveGenerationalCheckpoint(generation, checkpoint_path):
    checkpoint = {
        "generation": generation,
        "student_state_dict": models[generation].state_dict(),
        "projections_state_dict": {
            gen_id: projections[gen_id].state_dict()
            FOR gen_id IN range(0, generation + 1)
        },
        "optimizer_state_dict": optimizers[generation].state_dict(),
        "config": {
            "teacher_model_id": "Qwen/Qwen1.5-1.8B",
            "student_model_id": generation_models[generation].model,
            "num_predecessors": generation,
            "kd_loss_weight": kd_loss_weight,
            "ce_loss_weight": ce_loss_weight,
            "common_dim": 768
        }
    }
    TORCH.SAVE(checkpoint, checkpoint_path)


LoadGenerationalCheckpoint(generation, checkpoint_path):
    checkpoint = TORCH.LOAD(checkpoint_path)
    
    models[generation].load_state_dict(checkpoint["student_state_dict"])
    
    FOR gen_id, proj_state IN checkpoint["projections_state_dict"].items():
        projections[gen_id].load_state_dict(proj_state)
    
    IF optimizers[generation]:
        optimizers[generation].load_state_dict(checkpoint["optimizer_state_dict"])


================================================================================
9. KEY DIFFERENCES: SINGLE vs GENERATIONAL
================================================================================

Single Distillation (Original):
    ├─ One teacher (frozen)
    ├─ One student (trainable)
    └─ Loss = KD_loss(student, teacher) + CE_loss(student)

Generational Distillation (New):
    ├─ One teacher (frozen, Gen 0)
    ├─ Multiple students (Gen 1, 2, ..., N)
    ├─ Each student learns from ALL predecessors
    ├─ Gen 1 loss = KD(Gen1, Gen0) + CE(Gen1)
    ├─ Gen 2 loss = KD(Gen2, Gen0) + KD(Gen2, Gen1) + CE(Gen2)
    ├─ Gen N loss = Σ KD(GenN, Gen0..N-1) + CE(GenN)
    └─ Accumulated knowledge flows through lineage


Benefits of Generational Approach:
    1. Each student benefits from multiple teacher signals
    2. Models progressively compress while maintaining performance
    3. Cross-generational knowledge transfer prevents information loss
    4. Can create generation chains: 1.8B → 0.5B → 0.3B → 0.1B
    5. Each generation is validated against all predecessors


================================================================================
10. TRAINING STRATEGY FOR GENERATIONS
================================================================================

SEQUENTIAL TRAINING (Recommended):
    
    STEP 1: Train Generation 1
        └─ Learns from: Teacher (Gen 0)
    
    STEP 2: Train Generation 2
        └─ Learns from: Gen 0 + Gen 1
    
    STEP 3: Train Generation 3
        └─ Learns from: Gen 0 + Gen 1 + Gen 2
    
    ... and so on


Loss Weight Distribution (Equal):
    Gen 1: weight_Gen0 = 1.0
    Gen 2: weight_Gen0 = 0.5, weight_Gen1 = 0.5
    Gen 3: weight_Gen0 = 0.33, weight_Gen1 = 0.33, weight_Gen2 = 0.33
    Gen N: weight_each = 1.0 / N


Alternative Loss Weight Distribution (Recency Bias):
    // Give more weight to more recent models
    Gen 2: weight_Gen0 = 0.3, weight_Gen1 = 0.7
    Gen 3: weight_Gen0 = 0.2, weight_Gen1 = 0.3, weight_Gen2 = 0.5

Cross-Generational Comparison:

                Gen 0   Gen 1   Gen 2   Gen 3   Gen N
Gen 1 vs Gen0   ✓KD     -       -       -       -
Gen 2 vs Gen0   ✓KD     ✓KD     -       -       -
Gen 2 vs Gen1   KD      ✓KD     -       -       -
Gen 3 vs Gen0   ✓KD     ✓KD     ✓KD     -       -
Gen 3 vs Gen1   KD      ✓KD     ✓KD     -       -
Gen 3 vs Gen2   KD      KD      ✓KD     -       -

Note: ✓KD = Primary distillation target during training
      KD = Secondary validation metric

Each generation is measured against:
    1. Immediate predecessor (most important)
    2. All previous generations (validation)
    3. CE loss on language task (anchor)
